# 小工具点子备忘

> 供后续选做功能时参考；与实现无关，可随时增删。

---

## 一、常见实用向（大路货，易做）

适合纯前端、GitHub Pages；实现快但同质化高。

### 文本与编码

- Base64 / URL 编解码；多行文本、文件拖入转 Base64（或反向）
- JSON 格式化 / 压缩 / 校验（错误行提示、一键复制）
- 哈希摘要：SHA-256、MD5（Web Crypto 等），对字符串或小文件
- UUID / 随机字符串生成（长度、字符集可选）
- 大小写、全半角、去空行等纯文本批处理

### 时间与数字

- 时间戳 ↔ 可读时间（本地时区、UTC、复制结果）
- Cron 表达式说明（对照表 + 简单解释）
- 进制转换：二 / 八 / 十 / 十六

### 颜色与样式

- 颜色转换：HEX / RGB / HSL
- 对比度检查：前景 + 背景，对照 WCAG

### 图片与媒体（客户端）

- 图片压缩 / 改尺寸（Canvas，不上传服务器）
- 二维码生成（解析需额外库）
- 上传图取色：点击得 HEX（`getImageData`）

### 开发与调试

- 正则测试：pattern + 文本高亮（注意 ReDoS，限制长度）
- User-Agent 解析展示
- JWT 仅解码 payload（Base64），**不验证签名**，明确勿粘贴敏感 token

### 生活向

- 番茄钟 / 倒计时（可选通知权限）
- 单位换算：长度、重量、温度等

### 实用补遗（第二波）

- 两列文本 **Diff**（逐行对比，导出 Markdown / 带样式 HTML，全本地）
- 粘贴乱表格 → **对齐成 Markdown / TSV**，可选去重行、按列排序
- **CIDR / 子网**：网络位、可用主机、广播、wildcard（附简短说明）
- **常用端口速查**：内置表 + 搜索（22、53、443、5432…）
- **字数 + 阅读时间**：可分中英文估算语速（如 300 / 450 字每分钟）
- **Luhn 校验**：演示卡号校验位算法，**严禁填真卡、不落库**，教育向免责
- **ISBN / EAN 校验**：格式与校验位说明
- 粘贴 **ICS** 片段 → 人类可读日程列表（RRULE 可先不做或只做简单重复）
- 粘贴 SVG **`path d`** → 小画布预览 + 大致宽高（bbox）
- 粘贴 **HTTP 响应头** 文本 → 高亮 `CSP`、`HSTS`、`X-Frame-Options` 等
- 粘贴 **日志** → 用正则抓时间戳，排序成简易时间线（可与「正则测试」联动思路）
- 标识符风格批量互转：`camel` / `Pascal` / `snake` / `kebab` / `CONST`
- **YAML ↔ JSON** 互转（与 JSON 工具并列；大文件限长防卡死）
- **`.env` 脱敏预览**：键保留、值替换为占位符，仅本地展示
- HTML **实体**编解码；可选「剥标签」保留纯文本
- **Slug 生成**：标题 → URL 友好串（多语言可简化为去特殊字符 + 连字符）

### 若只想快速加一个实用工具

优先考虑：JSON 格式化、Base64、时间戳、HEX/RGB、简单图片压缩。

## 二、脑洞 / 有意思向（差异化）

偏好玩、记忆点强；多数仍可主要靠浏览器完成。

### 交互与感官

- **「今天有多亮」计**：`AmbientLightSensor`（若支持）或摄像头画面平均亮度 + 吐槽文案
- **摇一摇决策器**：`DeviceMotion` 检测摇晃，随机抽选项
- **嘴型同步假唱（整蛊）**：摄像头 + 极简「开合」检测 + 字幕生成器
- **键盘 ASMR 模式**：按键触发波形/粒子（可关声）

### 无厘头生成

- **赛博签文**：生日或随机种子 → 黑话运势 + 伪周易卦象（SVG）
- **废话文学扩写器**：一词递归扩写成「说了等于没说」长文
- **公司黑话翻译机**：人话 ↔ 周报体（规则 + 随机槽位）
- **物种鉴定（整蛊）**：任意图 → 假「学名」与栖息地（本地随机表 + 免责说明）

### 时间与存在

- **人生进度条**：生日 →「人生的第几周」；可按预期寿命切换
- **倒计时到下一个质数秒** 或 **回文时间**（数学小趣味）
- **时区叠罗汉**：多城市「谁在上班谁在睡」简易时间条

### 视觉与物理感

- **桌面鱼缸**：CSS/Canvas 小鱼跟鼠标或躲避
- **窗户雨声合成器**：滑块调雨强、雷距、空调白噪（Web Audio）
- **像素沙漏**：数字沙一格格落，可选音效

### 社交与分享（可不登录）

- **一句话密文**：句子和密码 → 前端加密成 emoji 串 / base64url，同页解密
- **默契测试生成器**：各本地作答，再「揭晓」对比（可简化为 hash 对比思路）

### 略硬核但怪

- **马尔可夫胡话**：内置语料，现场生成下一句
- **康威生命游戏**：上传头像二值化当初始图案

### 脑洞升格（更飞一点）

- **宇宙日历**：把宇宙史压成「一年 365 天」，显示此刻对应宇宙历的几月几日几时（科普 + 存在感）
- **缸中之脑自检**：假哲学问卷 → 输出一本正经的扯淡「检测报告」与证书图
- **外星信号破译（整蛊）**：任意文本 → 假 emoji 密文 + 假星系坐标 + 免责
- **时间是否为模拟**：一键生成「定理」式胡说 Markdown（伪学术排版）
- **善恶天平抽象艺术**：左右拖善行/恶行滑块，Canvas 天平歪成不可名状
- **向平行宇宙借寿命**：荒诞计数器，点一下「借一分钟」，文案越来越离谱
- **若地球自转急停**：纯科普向夸张滑杆 + 配图文案（非真物理仿真，避免误导可脚注）
- **太阳系若你是蚂蚁**：输入身高 → 日月地等「若等比缩小」成日常物体比喻
- **URL 咒文化**：把链接字符映射成卢恩/符文风展示 + 可选 Web Speech「念咒」
- **你的脑内分辨率**：摄像头（可选）+ 整蛊问卷 → 输出「渲染管线」梗图（娱乐向）
- **电子木鱼功德证书**：计数、音效、一键导出梗图（与站点「玄学」工具线一致）
- **Tab 标题电子宠物**：只靠改 `document.title` 显示饿/睡/开心（极简状态机）
- **SemVer 占星**：输入 `x.y.z` 生成今日签文（与赛博签文同系列）
- **Commit 灵气鉴定仪**：分析 message 长度、conventional、emoji 密度 → 玩梗评分

### 脑洞再升格（轨道外 · 概念优先）

> 多数「像产品 demo + 段子」即可，不必真仿真；**免责与伪科学标注**写大字。

- **多 Tab 量子纠缠**：`BroadcastChannel` 连两个窗口，点「测量」随机塌缩成同款梗图/文案（伪量子）
- **阿卡西记录检索机**：输入任意词 → 生成巴洛克风「宇宙档案条目」（假条形码、假分类号、假签发机构）
- **旧日支配者招聘 JD**：选职位模板 → 输出克系 HR 文案 + 离谱福利 + 面试诅咒条款
- **浏览器在做梦**：每次点击累加「梦层数」，页面叠化越来越 surreal 的滤镜/漂浮元素（性能限流）
- **你的个人热寂倒计时**：用热力学梗文案 + 随机「熵增事件」推送（非科学预测，纯玩笑）
- **人形加载条**：摄像头人脸 → 切成环形进度条纹理或 CSS mask（需权限；可只做上传图）
- **桌面黑洞领养**：拖 emoji「物质」进事件视界，喷霍金辐射粒子（Canvas/WebGL 简版）
- **代码月相占卜**：粘贴目录树或文件 hash 摘要 → 输出「满月/新月」与扯淡宜忌（不写死真迷信）
- **两个宇宙合并公投**：输入双方「国名/种子」→ 生成一部胡编宪法序言 + 国旗色条
- **薛定谔的发布按钮**：未 hover 前同时显示绿/红（纯 CSS），hover 才「观测」结果
- **希尔伯特旅馆入住**：输入姓名 → 无限旅馆叙事段落（搬房间、加一客人等梗）
- **回形针最大化迷你版**：点击产回形针，UI 逐渐被「回形针」文案吞噬（致敬思想实验，点到为止）
- **曼德博罗 aura**：鼠标轨迹用逃逸时间着色，宣称「今日分形气场」（数学可视化 + 玄学旁白）
- **这个像素有意识吗**：全屏一个像素 + 投票条 + 实时假统计图（哲学恶搞）
- **DNS 神谕**：输入域名（或假解析）→ 把 TLD/长度映射成卦辞式句子（可不接真实 DNS，纯规则）
- **Service Worker 守護霊**：展示 SW 注册状态 =「灵压条」+ 神社 UI（技术状态娱乐化）
- **把 localStorage 当骨灰龛**：键名像牌位，值写寄语，一键「超度」清空（黑色幽默，慎用文案）
- **世界线收束率**：输入两件小事 → 输出假 STEINS;GATE 风百分比 + 胡扯世界线描述（二创致敬，版权注意）
- **费米悖论你的锅**：选「我为什么没见到外星人」→ 生成把责任推给你的荒谬论证链
- **降维打击预览器**：3D CSS 方块 → 滑杆压成 2D 再成 1D 线（视觉梗，附「不要当真」）

### 若只做一个「有人设」的

优先：**赛博签文**、**人生第几周**、**窗户雨声合成器**（与「摄像头镜子」一样容易让人记住站点调性）。

## 三、落地时注意

- 摄像头、传感器、通知等多为 **HTTPS 或 localhost** 下才稳定可用
- 整蛊类务必有 **免责 / 娱乐说明**，避免被当真
- 加密类工具要写清 **密钥与数据不离开本机**（若确实如此）
