# AI 开发文档

本目录用于存放与 AI 辅助开发相关的所有文档，包括指导说明、开发规范以及改动记录。

## 目录结构

```
ai/
├── README.md          # 本文件，目录总览
├── guides/            # 指导说明
│   ├── project-context.md
│   ├── dev-tasks.md   # 开发任务计划与勾选
│   ├── tool-ideas.md  # 小工具点子备忘（实用向 + 脑洞向）
│   └── ...            # 其他规范、提示词模板等
└── changelog/         # 开发改动记录
    └── ...            # 每次 AI 辅助改动的记录文件
```

## guides（指导说明）

存放与 AI 协作相关的指导文档，例如：

- 项目背景与技术栈说明，供 AI 快速了解上下文
- 编码规范与风格要求
- 常用提示词模板
- AI 使用注意事项
- 小工具功能灵感：[`guides/tool-ideas.md`](guides/tool-ideas.md)

## changelog（开发改动记录）

按日期或功能模块记录每次 AI 辅助开发的改动，例如：

- 新增功能说明
- 重构内容
- Bug 修复
- 相关决策与背景

命名建议：`YYYY-MM-DD-<功能描述>.md`，如 `2026-03-25-add-ai-docs.md`
