# 小工具集

实用小工具集合网站，基于 Vue 3 + Vite 开发。

## 功能

- **甘特图** - 拖拽调整任务工期，支持自定义日期范围
- **掷筊** - 模拟圣杯/笑杯/阴杯，支持摇一摇
- **摄像头镜子** - 调用前置摄像头镜像显示
- **三体模拟** - Canvas 万有引力运动模拟
- **人形进度环** - 将照片脸部裁成环形进度扇区

## 技术栈

Vue 3 + TypeScript + Vite + Pinia + Vue Router

## 开发

```sh
pnpm install
pnpm dev
```

## 构建

```sh
pnpm build
pnpm preview
```

## 测试

```sh
pnpm test:unit
pnpm test:e2e
```

## Lint

```sh
pnpm lint
```
